# Proyecto 01
# Cluster de K-Means de sismos
# Este proyecto muestra la aplicación de un modelo de aprendizaje no supervisado utilizando el modelo de K-Means, donde se hace una agrupación
de clústeres determinada por el método del codo, logrando segmentar la actividad sismica sin distinción de etiquetas, basándose en la relación intrinseca entre la latitud y la profundidad con el objetivo de entender el funcionamiento de dicho modelo.

# Se aplicó un tratamiento de análisis de datos para identificar valores nulos y outliers, asegurándome de que la información no fuera sesgada ni los valores con nulos afectaran la agrupación del clúster, esto porque el modelo de K-Means es muy sensible a valores nulos y outliers (valores extremos) aplicando fórmulas matemáticas como el logaritmo base 10 para la transformación de los datos (suavizado de extremos), el escalamiento de los datos con puntuación z-score para calcular la distancia euclidiana, esto último para aplicar el método de K-Means.
